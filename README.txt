You must have paxrunner in your ath to run these examples. The best way to install is download the assembly from maven : http://search.maven.org/#search%7Cgav%7C1%7Cg%3A%22org.ops4j.pax.runner%22%20AND%20a%3A%22pax-runner-assembly%22

Download and unpack the assembly (using jar or a zip client) and make sure the bin dir is in your path.
